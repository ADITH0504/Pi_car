# Pi_car
This project uses a Raspberry Pi and an Arduino to work together over serial communication. The Pi handles the web page, streams live webcam video, and lets you control the system through a browser. The Arduino receives those commands from the Pi and controls the RC hardware.
Right now, the project includes the basic prototype: web hosting, live camera tracking, RC controls, and Pi‑to‑Arduino serial communication. More advanced features will be added in future updates.
