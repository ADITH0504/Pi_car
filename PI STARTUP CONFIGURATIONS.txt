PI STARTUP CONFIGURATIONS:

i) WHEN USING A NEW PI OR OS ALWAYS START WITH:
   sudo apt update
   sudo apt upgrade
   THIS WILL SET THE SYSTEM UP TO DATE


1. CHECK PYTHON 3 IS INSTALLED BY:
   python3 --version
   IF NOT UP TO VERSION 3, INSTALL IT BY:
   sudo apt install python3 python3-pip -y

2. DOWNLOAD VS CODE:
   * DOWNLOAD EXTENSION: PYTHON
   * CREATE A FOLDER FOR OUR PROJECT
   * CREATE A VIRTUAL ENVIRONMENT:
         python3 -m venv venv
         source venv/bin/activate
     (DO THIS BEFORE CREATING ANY FILE UNDER THE FOLDER)
   * SELECT THE VIRTUAL ENV INTERPRETER USING CTRL+SHIFT+P
   * CREATE THE FILE "main.py"
   * WRITE A SAMPLE PRINT FUNCTION, THEN SAVE IT AND RUN IT IN THE TERMINAL:
         python3 filename.py
   * DOWNLOAD THE LIBRARIES NEEDED FOR THE PROJECT (EXAMPLE):
         pip install pyserial

3. YOU ARE GOOD TO GO


i) CHECK THE USB PORT CONNECTED TO THE PI:
   ls /dev/tty*


FLASK INSTALLATION CODES:

   sudo apt update        (UPDATES THE LIST OF AVAILABLE PACKAGES)
   sudo apt install python3-pip -y   (INSTALLS THE PIP INSTALLER)
   pip3 install flask     (INSTALLS FLASK USING PIP3)

ii)WHAT IS PIP?
   PIP IS THE PACKAGE INSTALLER FOR PYTHON.


iii)PI STARTUP PROGRAM RUN:

- FIRST RUN THE FILE PATH IN TERMINAL TO CONFIRM IT IS CORRECT
- THEN:
      sudo crontab -e
- IF YOU SEE "SELECT AN EDITOR", CHOOSE THE FIRST ONE
- WRITE THE FOLLOWING COMMAND AT THE BOTTOM OF THE CRONTAB:
      @reboot python3 /home/Nathan/Desktop/Rc_project/app.py &

- SAVE AND EXIT BY PRESSING: CTRL+S THEN CTRL+X

iv)TO STOP THE PROGRAM FROM RUNNING:
      ps aux | grep python
     -IF YOU FIND THE FILE, NOTE THE NUMBER, THEN:
      kill XXX
      (XXX IS THE NUMBER YOU NOTED), ALSO TO NOT IF THE FILE IS IN ROOT KILL IT BY "sudo kill xxx"
